Super Hero

Aplicacion web desarrollada con React Js donde el usuario se logea y puede iniciar la busqueda de heroes para completar su equipo con 3 super heroes malos y 3 super heroes buenos. 
En la sección "mi equipo" podrá ver las características principales de su equipo además de ver el detalle de cada heroe y la posibildad de eliminarlos.

Librerias utilizadas:
#Bootstrap
#Formik
#Axios
#React-Bootstrap
